# claude-ai-methodology

**Claude(Anthropic) 관점에서 본 "AI를 가장 잘 쓰는 방법론"** — 딥리서치 기반 단일 페이지 리포트.

방법론의 핵심 개념(컨텍스트 엔지니어링·서브에이전트·하네스·오케스트레이션·검증 루프)을 Claude Code / Agent SDK의 **실제 기능에 1:1로 매핑**해 해설합니다. 모든 주장에 클릭 가능한 출처 링크 포함.

## 🌐 배포

GitHub Pages로 배포됨 → `index.html` (저장소 루트).

## 생성 방식

Claude Code 딥리서치 하네스로 생성:
- 5각도 병렬 검색 · 22개 소스 fetch · 105개 주장 추출
- 3표 적대적 검증(2/3 반증 시 폐기) → 23개 통과 / 2개 기각
- 104개 에이전트 가동

검증에서 **기각된 "멀티에이전트 90.2% 우위" 수치는 의도적으로 제외**했습니다. (vendor 내부 eval 단일 출처)
